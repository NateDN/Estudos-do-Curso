## 🔷 Introdução

Em muitas aplicações reais, os dados que um programa utiliza precisam ser armazenados e recuperados entre execuções. Por exemplo, um programa de cadastro de clientes precisa guardar os dados inseridos para acessá-los posteriormente. É aí que entra a **manipulação de arquivos**.

Python oferece ferramentas nativas para trabalhar com arquivos nos formatos mais comuns, como:

- **TXT**: arquivos de texto simples.
- **CSV**: planilhas com dados separados por vírgula.
- **JSON**: formato de dados estruturado muito usado em sistemas web e APIs.

## 🔹 1. Manipulação de Arquivos TXT

### O que é um arquivo TXT?

Arquivos `.txt` são arquivos de **texto puro**. São usados para guardar anotações, logs, listas e qualquer outro conteúdo textual.

### Abertura de arquivos

O Python usa a função `open()` para abrir arquivos.

**Sintaxe:**

```python
open("nome_do_arquivo.txt", "<modo>")
```

Modos comuns:

- `"r"`: leitura
- `"w"`: escrita (sobrescreve o conteúdo anterior)
- `"a"`: escrita no final do arquivo (append)

### Escrevendo em um arquivo TXT

```python
with open("mensagem.txt", "w") as arquivo:
    arquivo.write("Olá, mundo!\n")
    arquivo.write("Este é um exemplo de escrita em arquivo.")
```

**Explicações:**

- O comando `with` abre o arquivo e o fecha automaticamente após o bloco.
- `"w"` indica que estamos escrevendo.
- `\n` quebra a linha.
- `arquivo` é uma **variável** que aponta para uma **instância do arquivo aberto**.
- É como dizer: *“Abra o arquivo e me dê um controle sobre ele com o nome `arquivo`.”*

### Lendo um arquivo TXT

```python
with open("mensagem.txt", "r") as arquivo:
    conteudo = arquivo.read()
    print(conteudo)
```

### Outras formas de leitura:

```python
# Ler linha por linha
with open("mensagem.txt", "r") as arquivo:
    for linha in arquivo:
        print(linha.strip())
```

O método **`strip()`** serve para **remover espaços em branco** (e quebras de linha) **do começo e do fim** de uma **string**.

## 🔹 2. Manipulação de Arquivos CSV

### O que é um arquivo CSV?

**CSV** significa **Comma-Separated Values**, que em português quer dizer **“valores separados por vírgulas”**. É um tipo de arquivo usado para armazenar informações em formato de tabela, parecido com uma planilha. Nesse formato, cada linha do arquivo representa uma linha da tabela, e os dados de cada coluna são separados por vírgulas. O formato CSV é muito utilizado porque é simples, fácil de abrir e funciona em diversos programas, como o Excel e o Google Planilhas.

Exemplo de arquivo CSV:

```
nome,idade,email
Ana,22,ana@email.com
João,30,joao@email.com
```

Python possui uma biblioteca chamada `csv` para trabalhar com esse tipo de arquivo.

### Escrevendo em arquivos CSV

```python
import csv

with open("dados.csv", "w", newline='', encoding="utf-8") as arquivo:
    escritor = csv.writer(arquivo)
    escritor.writerow(["nome", "idade", "email"])  # Cabeçalho
    escritor.writerow(["Carlos", 25, "carlos@email.com"])
    escritor.writerow(["Maria", 28, "maria@email.com"])
```

**Explicações:**

- Cada `linha` é uma lista com os valores separados por vírgula.
- `newline=''` evita problemas com quebras de linha em diferentes sistemas operacionais.
- `encoding="utf-8"` diz ao Python para gravar o arquivo usando a codificação **UTF-8**, garantindo que **acentos e caracteres especiais** sejam salvos corretamente.
- `csv.writer(...)` cria um objeto escritor de arquivos CSV.
- `escritor.writerow(...)` escreve uma linha no arquivo CSV.

### Lendo arquivos CSV

```python
import csv

with open("alunos.csv", newline='', encoding='utf-8') as arquivo:
    leitor = csv.reader(arquivo)
    for linha in leitor:
        print(linha)
```

**Explicações:**

- Cada `linha` é uma lista com os valores separados por vírgula.
- **`csv.reader(...)`**: cria um **objeto leitor**, que lê o conteúdo do arquivo CSV linha por linha, separando os valores onde houver vírgulas.

### Lendo CSV como dicionário

```python
import csv

with open("alunos.csv", newline='', encoding='utf-8') as arquivo:
    leitor = csv.DictReader(arquivo)
    for linha in leitor:
        print(f"{linha['nome']} tem {linha['idade']} anos.")
```

Essa forma permite acessar os dados pelas **chaves do cabeçalho**, o que torna o código mais legível.

## 🔹 3. Manipulação de Arquivos JSON

### O que é JSON?

**JSON** (JavaScript Object Notation), que em português significa **"Notação de Objetos JavaScript"**, é um formato muito utilizado para **armazenar e trocar dados estruturados**. Ele é leve, fácil de ser lido tanto por humanos quanto por máquinas, e é o formato padrão utilizado em muitas APIs (interfaces de comunicação entre sistemas).

Em Python, um JSON é muito parecido com um **dicionário**, pois utiliza pares de chave e valor para organizar os dados.

### Exemplo de JSON:

```json
{
  "nome": "Alice",
  "idade": 24,
  "cursos": ["Python", "HTML", "Excel"]
}
```

### Salvando um dicionário em JSON

```python
import json

dados = {
    "nome": "Alice",
    "idade": 24,
    "cursos": ["Python", "HTML", "Excel"]
}

with open("dados.json", "w", encoding="utf-8") as arquivo:
    json.dump(dados, arquivo, indent=4)
```

- `import json`: importa o módulo `json`, que permite trabalhar com dados no formato **JSON**
- `json.dump()`: função que **salva o dicionário no arquivo**, convertendo-o para o formato JSON.
- `arquivo`: é o destino onde os dados serão escritos.
- `indent=4`: deixa o arquivo formatado com indentação de 4 espaços, tornando-o mais **legível**.

### Lendo dados de um arquivo JSON

```python
import json

with open("dados.json", "r", encoding="utf-8") as arquivo:
    conteudo = json.load(arquivo)
    print(conteudo["nome"])
    print(conteudo["cursos"])
```

### Convertendo entre dicionários e JSON

```python
# Dicionário → JSON (string)
texto_json = json.dumps(dados)
print(texto_json)

# JSON (string) → Dicionário
novo_dado = json.loads(texto_json)
print(novo_dado["idade"])
```
